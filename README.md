# Heart-Rate-Monitor
Simple cross platform application to display heartrate, oximetry received from esp8266 using max30100

# hardware Setup
in this project i'm using MAX30100 

## Wire

| MAX30100 | ESP8266 |
|----------|---------|
| GND      | GND     |
| VIN      | 3V3     |
| SCL      | D1      |
| SDA      | D2      |
